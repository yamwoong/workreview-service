## 📄 1. README.md

### 🎯 역할
**프로젝트의 시작점이자 모든 문서를 연결하는 허브**

### 📌 주요 내용
```
✓ 문서 목록 및 우선순위
✓ Cursor + Claude 활용 전략
✓ 개발 워크플로우 (단계별 가이드)
✓ 효과적인 프롬프트 작성법
✓ FAQ & 문제 해결
✓ 체크리스트
```

### 🌟 특징
- **종합 가이드**: 다른 모든 문서를 어떻게 활용할지 설명
- **실전 예시**: Cursor Composer, Claude Chat 사용 예시 포함
- **단계별 워크플로우**: Phase 0(준비) ~ Phase 2(추가기능)까지 구체적 일정
- **프롬프트 템플릿**: 좋은 프롬프트 vs 나쁜 프롬프트 비교

### 💡 언제 사용?
- **프로젝트 시작 전**: 가장 먼저 읽어야 할 문서
- **막힐 때마다**: FAQ, 문제 해결 섹션 참고
- **팀 온보딩**: 새 팀원에게 가장 먼저 공유
